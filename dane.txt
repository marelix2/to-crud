 temat projektu: 

	38. REST Web Services CRUD
	Automatyczne generowanie na podstawie metadanych pobranych z bazy danych usług REST Web Services pozwalających na tworzenie, odczytywanie, aktualizację i kasowanie danych (CRUD). Należy 		opracować stronę internetową pozwalającą na uruchamianie tych usług.


 spis technologii, które będą używane:

	-backend: 
	  Node.js (środowisko uruchomieniowe)
	  Express.js (framework do obsługi zagadnień związanych z komunikacją)
	  http-server ( bliblioteka wspierająca protokół http)
	  express-routes-mapper ( konfiguracja ścieżek, lepsza współpraca z protokołem http)
	  sqlite3 (wsparcie dla bazy danych sqlite)

	-baza danych: 
	  Sqlite3 (relacyjna baza danych)

	-frontend: 
	  JavaScript (ES6+)
	  React.js v16.8 ( framework js)
	  lodash (biblioteka do ułatwienia pracy nad obiektami, tablicami)
	  antd ( framework ze stylami)
	  scss ( preporcesor styli css)
	  axios ( bilbioteka wspierająca proktokuł http)
